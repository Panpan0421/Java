# 大小端

**大端模式**：就是高位字节排放在内存的低地址端，低位字节排放在内存的高地址端。

**小端模式**：就是低位字节排放在内存的低地址端，高位字节排放在内存的高地址端。

例如数字 0x12 34 56 78

**1)大端模式：**

低地址 -----------------> 高地址

0x12  |  0x34  |  0x56  |  0x78

**2)小端模式：**

低地址 ------------------> 高地址

0x78  |  0x56  |  0x34  |  0x12

**互相的优势：**

小端模式 ：因为高地址存放高位，强制转换数据不需要调整字节内容
大端模式 ：低地址存放符号位，很容易判断正负
